# Budget Tracker

## Purpose
A simple and efficient web application to manage personal finances by tracking income and expenses. This app helps users understand their spending habits through real-time calculations and visual charts.

## Features
* **Add Transactions**: Input description and amount, then choose Income or Expense.
* **Live Calculations**: Automatically updates Total Income, Total Expenses, and Balance.
* **Visualization**: A dynamic bar chart showing the percentage of income vs. expenses.
* **Filtering**: View all transactions or filter them by type (Income/Expense).
* **Data Persistence**: Uses LocalStorage to save data so it remains after refreshing the page.
* **Delete Function**: Easily remove any transaction from the list.

## Technologies Used
* **HTML5**: Page structure.
* **CSS3**: Dashboard layout using Grid and Flexbox.
* **JavaScript**: Logic for calculations, storage, and UI updates.
* **FontAwesome**: Icons for a professional look.

## Setup Instructions
1. Save the `index.html`, `style.css`, and `script.js` files in the same folder.
2. Open the `index.html` file in any modern web browser.
3. Your data will be saved locally in your browser automatically.
